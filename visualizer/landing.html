How It Works

Three steps to complete clarity

Get from zero to full security visibility in under a minute.

1

Install

One command to install. Works with npm, pip, or Docker.

npm install -g slop-auditor
2

Scan

Point it at any repo or AWS account. All scanners run in parallel.

slop-auditor scan ./my-project
3

Visualize

Open the 3D dashboard. See every finding, prioritized and actionable.

slop-auditor serve && open /app
