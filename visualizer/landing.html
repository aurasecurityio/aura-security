How It Works

Three ways to run

CLI for quick scans, CI for automation, or service mode for real-time monitoring.

1

Install

Install globally via npm or pip, or run with Docker.

npm install -g slop-auditor
2

Scan

Point it at any directory or AWS account. Results in seconds.

slop-auditor scan ./my-project
3

Visualize

Open the 3D dashboard for an interactive view of findings.

slop-auditor serve && open /app
